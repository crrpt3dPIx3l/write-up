# Quantum Scrambler
## Description
We invented a new cypher that uses "quantum entanglement" to encode the flag. Do you have what it takes to decode it? Connect to the program with netcat: $ nc verbal-sleep.picoctf.net port The program's source code can be downloaded here.

### Hints
1. Run eval on the cypher to interpret it as a python object
2. Print the outer list one object per line
3. Feed in a known plaintext through the scrambler

## Solution
Started by downloading the file provided in the challenge the examine the content of the file
```
import sys

def exit():
  sys.exit(0)

def scramble(L):
  A = L
  i = 2
  while (i < len(A)):
    A[i-2] += A.pop(i-1)
    A[i-1].append(A[:i-2])
    i += 1
    
  return L

def get_flag():
  flag = open('flag.txt', 'r').read()
  flag = flag.strip()
  hex_flag = []
  for c in flag:
    hex_flag.append([str(hex(ord(c)))])

  return hex_flag

def main():
  flag = get_flag()
  cypher = scramble(flag)
  print(cypher)

if __name__ == '__main__':
  main()
```
